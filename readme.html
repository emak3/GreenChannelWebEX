概要 Overview

グリーンチャンネルWeb のライブ配信ページで、各チャンネルの映像を別ウィンドウに切り出して表示する拡張機能です。

非公式：JRA・グリーンチャンネル公式の提供物ではありません。利用規約・著作権は公式サイトの定めに従ってください。

主な機能 Features

  • サブウィンドウで開く：1ch〜5ch のタブ付近にボタンを追加し、選択チャンネルをポップアップ型ウィンドウで開きます。
  • 映像中心表示：別ウィンドウではヘッダー・フッター等を隠し、プレイヤーを画面いっぱいに近い形で表示します。
  • 配信休止時：サイト側でチャンネルが利用不可のとき、ボタンを無効化します。
  • タブバー削減：chrome.windows の popup 型で開きます。

インストール Install

  1. chrome://extensions（Edge は edge://extensions）を開く
  2. デベロッパーモードをオン
  3. パッケージ化されていない拡張機能を読み込むでフォルダ grcEx を指定

ファイル構成 Files

  • grcEx/manifest.json … 拡張定義
  • grcEx/content.js … UI・チャンネル同期
  • grcEx/background.js … ポップアップウィンドウ作成
  • grcEx/inject.css … 表示調整
  • gch-pages.css … 本ページ・プライバシー用スタイル
  • privacy.html … プライバシーポリシー
